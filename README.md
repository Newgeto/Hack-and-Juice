# Hack-and-Juice

Projet d’étude réalisé dans le cadre d’Epitech autour de la **sécurité applicative / pentest**.

L’objectif est de regrouper des notes, exemples et supports de cours autour des vulnérabilités web les plus courantes (inspiration OWASP), afin de comprendre :

- comment elles apparaissent,
- quels risques elles impliquent,
- et comment les prévenir/corriger.

## Collaboration

Travail réalisé sur Google Colab avec `Gogo7895`.

## Structure

Le dépôt est organisé par thématiques :

- `Broken Access Control`
- `Broken Anti Automation`
- `Broken Authentication`
- `Cryptographic Issues`
- `Improper Input Validation`
- `Injection`
- `Security through Obscurity`
- `Sensitive Data Exposure`
- `XSS`

Des ressources additionnelles peuvent se trouver dans `img-au-cas-ou`.

## Usage

Chaque dossier contient du contenu lié à la catégorie (exemples, explications, captures, etc.).
