# Injection - 3 - Database Schema

## Outil utilisé

* **Burp Suite Community** utilisé pour intercepter et analyser les requêtes HTTP

### Méthodologie

J’ai recherché un article sur le site, puis j’ai intercepté la requête avec Burp Suite, dans le paramètre de recherche j’ai remplacé la valeur par du texte au hasard, ensuite avec l’aide de vidéos et de sites trouvés en ligne j’ai modifié la requête, ce qui m’a permis d’obtenir la base de données directement dans la réponse Burp.
